# 熊猫栏舍建造引导流程

## 完整流程

```
序章 → 第一章 → 解锁小熊猫图鉴 → 图鉴引导 → 第二章 → 建造引导 → 玩家点击建造 → 建造中（2秒）→ 建造完成
```

### 1. 剧情阶段

1. 玩家完成序章，自动进入第一章
2. 第一章结束，解锁小熊猫图鉴，设置待返回故事为第二章
3. 图鉴引导完成后，自动播放第二章
4. 第二章结束，标记 `storyFlags['第二章'] = true`，返回动物园主页

### 2. 建造引导

第二章结束回到主页后，栏舍区域显示建造引导，提示玩家点击建造。

### 3. 玩家点击建造

玩家点击栏舍区域，触发建造流程，显示"建造中"提示，持续约 2 秒。

### 4. 建造完成

建造中提示结束后，同时显示已建成的 1 级栏舍和建造完成庆祝效果。

---

## 触发条件

建造引导需要以下条件全部满足：

1. 栏舍为 `red-panda-grove`
2. 栏舍未解锁且未在建造中
3. 没有图鉴后续引导（`pendingGuideSpeciesId` 为空）
4. 没有待返回的故事（`pendingReturnStoryId` 为空）
5. 第二章剧情已完成（`storyFlags['第二章']` 为 true）

### 不显示的情况

- 第二章未完成 → 栏舍区域不渲染
- 图鉴引导进行中 → 图鉴引导优先
- 第二章待播放 → 等第二章播完
- 栏舍已解锁或正在建造 → 不显示引导
